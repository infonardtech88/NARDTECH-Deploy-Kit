# NARDTECH-Deploy-Kit
NARDTECH IT Toolkit è uno script PowerShell che installa in automatico i software essenziali per PC Windows (browser, Office, PDF, utility) più strumenti di diagnostica IT (scanner di rete, monitoraggio hardware, sicurezza). Scegli tu cosa installare: tutto, solo l'essenziale, o a scelta. Firmato NARDTECH.
